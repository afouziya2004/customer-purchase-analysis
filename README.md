# Customer Purchase Behavior Analysis & Segmentation

This project analyzes real-world e-commerce transaction data
to understand customer purchasing behavior and segment customers
based on revenue contribution.

## Dataset
Online Retail Dataset from Kaggle (downloaded automatically using kagglehub).

## Key Questions Answered
- Who are high-value customers?
- Which countries generate the most revenue?
- How can customers be segmented based on spending?

## Skills Demonstrated
- Data cleaning
- Business analytics
- Customer segmentation
- NumPy-based numerical analysis
- Data visualization

## Tech Stack
Python, Pandas, NumPy, Matplotlib, kagglehub
